Emissions

? This Agent will be able to update the emission of CO2 data for the power plants data in the world using 2 different models, the surrogate model and factor model. The factor model will calculate the CO2 emission value based on the capacity of the individual plant and the fuel type. For the surrogate model, the CO2 emission will be calculated by several paramaters like the annual generation, capacity, fuel type, year of the plant built and the technology used. Thus, it will provide more accurate calculation compared to the factor model. The number of agent involved will determined how many agents of specified model will be chosen to do the task. The number of plant node will determine how many plants will be modified by the agents. if you typed "all" in the number of plant node, it will modify all plants related to the "world power plants" top node.

Basics:

Electricity Cost  = Resulted Electricity Consumption × Electricity unit cost Gas Cost  = Resulted Gas Consumption × Gas unit cost

Example input:
query = {
'input':{'Resulted Electricity Consumption': 10 ,
'Resulted Gas Consumption': 10
'Electricity unit cost': 0.5
'Gas unit cost': 0.05
}
}

Arguments:
  • Resulted Electricity Consumption: refer to resulted energy consumption calculation agents for details

  • Resulted Gas Consumption: refer to resulted energy consumption calculation agents for details

  • Electricity unit cost:Put the value in the docker-compose file, in the unit of £/kWh


  • Gas unit cost:Put the value in the docker-compose file, in the unit of £/kWh


    • Example return:
    return = {
    'Electricity Cost': 100,
    'Gas cost': 100
    }

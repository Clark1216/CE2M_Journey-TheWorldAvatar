Basics:

Change of gas consumption = uptake × gas consumption × propotion of heating
Change of electricityconsumption  =
(boiler efficiency × change of gas)
COP
Resulted Gas consumption = Gas consumption - Change of gas consumption
Resulted Electricity consumption = Electricity consumption + Change of electricity consumption

NOTE: builded python function to call this agent can be found on: call_change_of_fuel_agent
Example Calculation:
query = {
'input':{'uptake': 0.5 ,
'gas consumption': 5
'electricity consumption': 5
'propotion of heating': 0.9
'cop': 2.5
'boiler efficiency': 0.8
'electricity consumption profile': [28.19,26.08,26.82,20.73,20.48,20.36,21.38,21.95,22.39,25.14,25.91,27.89]
'gas consumption profile': [7.88,7.54,7.54,4.86,4.14,3.78,3.78,3.64,4.05,6.09,6.74,8.46]
}
}

Arguments:
  • uptake: a index which quantify the extent of the deployment of heat pump by measureing the gas being replaced by using heat pump comparing to conventional fossil fule heating system.
    Defined as (change of gas / gas used for heating), or (change of gas / (total gas consumption * propotion of heating))

  • gas consumption: Value should be in unit of kWh

  • electricity consumption: Value should be in unit of kWh

  • propotion of heating: the propotion of gas which is used for heating(see equation above), if not provided, 0.9 will be used as default value.

  • cop: Coefficient of Performance.

  • boiler efficiency: boiler efficiency (see equation above), if not provided, 0.8 will be used as default value.

  • electricity consumption profile: A list containing 12 data represents the electricity consumption value from Jan to Dec

  • gas consumption profile: A list containing 12 data represents the gas consumption value from Jan to Dec


    Example return:
    return = {'change of gas': 100 ,
    'change of electricity': 100
    }
